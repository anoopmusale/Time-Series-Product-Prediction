# Time-Series-Product-Prediction
Quantity of each of the 100 key products sold in 118 days is given as the training data. Along with list of the product ids.  
# Implemented RNN
  Model: LSTM
  Layers: 7
  Units: 70/Layer
  Dropout: 20%
  Optimizer: adam
  Epoch: 2000
  Batch: 24
# Result:
Converges to approx. 1% Loss
